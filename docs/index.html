Feature breakdown

Selection popover

Highlight any epoch value (seconds or milliseconds) on a webpage. A compact popover appears with GMT, Local, and Relative time.

Epoch → Date

Paste or type an epoch value and see GMT/Local time plus a relative “ago/from now” display. Includes Epoch (ms) output.

Date → Epoch

Enter date and time fields (Local or GMT) and get Epoch (ms), Epoch (s), and Relative output instantly.

Relative → Epoch

Build timestamps relative to now using days/hours/minutes/ seconds/milliseconds and convert to epoch and dates.

History & privacy

The last 10 conversions are stored locally for quick access. Clear history anytime. No data leaves your device.
